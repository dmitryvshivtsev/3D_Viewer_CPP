3D_Viewer

About program

This program is designed to display and control 3D objects (files with the extension .obj). It has flexible settings, the ability to save images and record gif-animation.

What can it do?

  • * Loading a wireframe model from an obj file;
  • * Translation the model by a given distance in relation to the X, Y, Z axes;
  • * Rotation the model by a given angle relative to its X, Y, Z axes;
  • * Scale the model by a given value;
  • * Projection change(parallel and central);
  • * Show information about the uploaded model(file name, number of vertices and edges);
  • * Allow setting up the type (solid, dashed),color and thickness of the edges, display method (none, circle, square), color and size of the vertices;
  • * Allow choosing the background color;
  • * Allow saving the captured (rendered) images as bmp and jpeg files;
  • * Allow recording small screencasts by a special button into gif-animation.

How to install?

To install the program, run the "make install" command.

How to uninstall?

To uninstall the program, run the "make uninstall" command.
